Задание
=======

1. (При необходимости) Посмотреть текущую версию `Node` на странице https://nodejs.org/en.
Загрузить `.zip` архив с `node` и `npm` соответствующей верси отсюда: https://nodejs.org/dist/, разархивировать в каталог проекта.

2. Установить необходимые зависимости проекта с помощью `npm install`.

3. Запустить пример.

4. С использованием примера, получить HTML страницу с прогнозом погоды по городу Москва с какого-либо сервера, предоставляющего прогноз погоды (http://pogoda.yandex.ru, http://gismeteo.ru и т.д.).

5. С помощью строчных функций вывести текущую температуру из параметра body.
